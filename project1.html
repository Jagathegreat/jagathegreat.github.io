Bus Management System

Project Overview

This Bus Management System allows users to book bus tickets, view available buses, manage routes, and check schedules. The system is built with Java to handle backend operations and provides a smooth user experience.

Features

  • Bus Route Management: Admin can add and modify bus routes.
  • Booking System: Users can book bus tickets for their desired route.
  • Schedule Display: View available bus schedules and availability.
  • User Authentication: Login and registration for users to manage their bookings.

Technologies Used

The Bus Management System is developed using the following technologies:

  • Java - For backend programming.
  • MySQL - For database management.
  • HTML, CSS, JavaScript - For frontend development.
  • Apache Tomcat - As the web server for deployment.
